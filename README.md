## File Organizer Script (Bash)

## Objective

This project was created to practice Bash scripting by automating file organization.

## What the script does

- Scans files in the current directory
- Identifies files extensions
- Create folders based on extensions
- Moves files into their respective folders
- Handles file without extension
- Generates a log file with executed actions
- Counts processed files

## How to use

 ```bash
 bash organizer.sh
 ```
